## 🚀 Featured Project

### ⚡ Wattwise — AI Energy Compliance Platform
> Helping French landlords navigate the Loi Climat & Résilience with AI-powered renovation plans

- 🤖 AI energy audit via **Llama API** — personalized renovation plan in seconds
- 💰 Automatic subsidy calculator (MaPrimeRénov, CEE, Éco-PTZ)
- 🔍 RGE-certified contractor search by postal code
- 📊 National **ADEME database** integration (millions of real DPE records)
- 🚨 Real-time compliance dashboard with regulatory alerts

**Stack:** `Spring Boot` `Angular 19` `PostgreSQL` `TailwindCSS v4` `Llama API` `JWT` `ADEME API`
